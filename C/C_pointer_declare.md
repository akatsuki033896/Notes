# 指向变量

```c
int a = 1;
int* p; // 指向 a 的指针
```

# 数组相关声明

存储指针的数组，意思是里面全都是指针

```c
int p[10]; // 数组
int* p[10]; // p 是存储类型为 int* 的指针，size 为 10 的数组

int arr[10];
int (*p)[10] = &arr; // p 是指向一个size为10的int类型数组的指针 指向 arr
```

# 二级指针

指向指针 `p` 的指针

```c
int** p;
```

# 函数

```c
int func(int); // 返回类型为 int，参数类型为 int 的函数 func
int (*p)(int) = &func; // p 是指向一个返回类型为 int，参数类型为 int 的函数的指针
int (*p)(int) = func; // 函数名在表达式中会退化为指针
```

# 复杂声明

从变量名处起，根据运算符优先级结合，一步一步分析。 `[]` 在c语言中属于后缀运算符和 `()` 等同为最高优先级，然后是 `*`

1. 先找变量名 / 函数名
2. 从离变量名 / 函数名最近的地方开始看是否为指针 / 数组 / 函数
3. 看数据类型是什么

```c
int* (*p(int))[3];
```

`*p(int)` 为指向函数 `p()` 的指针，函数 `p()` 返回一个指针，指针指向一个存放类型为 `int*` 指针的数组，size 为 3

```c
int (*p)(int);
```

`*p` 是指针，指向了一个参数类型为 `int` 的函数，该函数返回类型为 `int`

## `typedef` 的使用

实在需要嵌套很多层的复杂声明，请使用 `typedef` 来肢解复杂的声明。将复杂的声明，先变为一个个简单的声明，然后取一个别名。

```c
int* (*p(int))[3];

typedef int* SB[3]; // 替换指向 int 类型 size 为 3 数组的指针类型为 SB
SB (*p(int)); // 函数 p(int) 返回一个指针，指针的类型是 SB
```
