C++ 中存储区域的分类：栈 / 堆 / 自由存储区 / 全局存储区 / 静态存储区 / 常量存储区

![[memory_model.png]]

# 代码 (.text)

代码区存放程序的二进制代码，只读，以防止程序在运行过程中被意外修改。

```cpp
#include <iostream>
int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
```

`main()` 的内容编译为二进制后，函数的逻辑就存放在代码区

# 全局 / 静态存储区

存储全局变量和静态变量，在 C 语言里已经初始化的变量放在 `.bss` 段，未初始化的放在 `.data` 段，但在 C++ 里没这个区分了，共同占有全局存储区

```cpp
#include <iostream>
int globalVar = 0; // 全局变量
void function() {
    static int staticVar = 0; // 静态变量
    staticVar++;
    std::cout << staticVar << std::endl;
}
int main() {
    function();
    function();
    return 0;
}
```

# 栈

栈存储函数调用时的局部变量，函数参数和返回地址，当函数调用完成后，分配给这个函数的栈空间会被释放。

`a`、`b`和`localVar`都是局部变量，它们存放在栈区，函数执行结束后释放

```cpp
#include <iostream>
void function(int a, int b) {
    int localVar = a + b;
    std::cout << localVar << std::endl;
}
int main() {
    function(3, 4);
    return 0;
}
```

# 堆

堆区是用于动态内存分配的区域，当使用`new`（C++）或者`malloc`（C）分配内存时，分配的内存块就位于堆区，需要手动使用 `delete` 和 `free` 释放，否则会内存泄漏

```cpp
#include <iostream>
int main() {
    int* dynamicArray = new int[10]; // 动态分配内存
    // 使用动态数组...
    delete[] dynamicArray; // 释放内存
    return 0;
}
```

# 常量区

用于存储常量数据，例如字符串字面量和其他编译时常量。这个区域通常也是只读的

```cpp
#include <iostream>
int main() {
	char* c="abc";  // abc在常量区，c在栈上。
  return 0;
}
```
