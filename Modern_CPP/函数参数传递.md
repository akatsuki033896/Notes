# 参数传递方式

## 值传递

将实参的值传递给形参，**函数内对形参的修改不会影响到实参**

```cpp
#include <iostream>

void swap_value(int a, int b) {
    int temp = a;
    a = b;
    b = temp;
}

int main() {
    int x = 10;
    int y = 20;
    swap_value(x, y); // x = 10, y = 20
}
```

## 引用传递

将实参的引用传递给形参。在这种情况下，函数内对形参的修改会影响到实参。

```cpp
#include <iostream>

void swap_reference(int &a, int &b) {
    int temp = a;
    a = b;
    b = temp;
}

int main() {
    ...
    swap_reference(x, y); // x = 20, y = 10
}
```

## 指针传递

将实参的地址传递给形参。在这种情况下，函数内对形参的修改会影响到实参。

```cpp
#include <iostream>

void swap_pointer(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

int main() {
    swap_pointer(&x, &y); // x = 20, y = 10
}
```

指针传递本质上也是值传递，只不过传递的值是内存地址而已

# 函数调用栈

一个函数调用时要在内存上存储参数和局部变量，但是多个函数调用时，需要保存别的东西保证这些函数正确返回，包括返回地址、`ebp`寄存器（基址指针寄存器，指向当前堆栈底部） 以及其它需要保存的寄存器。

```cpp
int func_a(int a, int *b) {
	a = 5;
	*b = 5;
};

int main() {
	int a = 10;
	int b = 10;
	func_a(a, &b);
	printf("a=%d, b=%d\n", a, b); // a = 10, b = 5
	return 0;
}
```

编译器会生成一段函数调用代码。将 `main` 函数内变量 `a` 的值拷贝到 `func_a` 函数参数 `a` 位置，将变量 `b`的地址，拷贝到 `func_a` 函数参数 `b` 的位置，**原理就是拷贝**
