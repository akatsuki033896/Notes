基本思想：基于C++11及之后的标准，实现生产者-消费者模型，包含一个队列，生产者生产的产品放入队列，消费者从队列中取出产品消费，对队列的访问是互斥的。分别将把产品放入队列，从队列取出产品这两个行为封装成函数 `void push_product(int i)` 和 `int pop_product()`，当推入序号为 `0` 的产品时说明不再向队列中推入产品。

```cpp
std::mutex mtx;
std::queue<int> q;

void push_product(int i) {
  mtx.lock();
  q.push(i);
  mtx.unlock();
}

int pop_product() {
  mtx.lock();
  auto product = q.front(); // 取出产品
  if (product != 0) {
      q.pop();
  }
  return product;
}

void producer() {
    for (int i = 1; i < 100; i++) {
        push_product(i);
    }
    push_product(0); // 发送标志位，表示结束
}

void consumer(int id) {
    while (int product = pop_product()) {
        std::cout << std::format("消费者 {} 得到了 {}\n", id, product);
    } // 不为0时
}
```

对互斥锁改进为使用符合RAII的  `unique_lock`，在作用域的 `}` 后自动析构，更安全，结合锁还可以使用条件变量 `condition_variable`，实现推入产品时通知消费者有新的产品

```cpp
std::mutex mtx;
std::condition_variable cv;
void push_product(int i) {
    {
        std::unique_lock<std::mutex> lk(mtx);
        q.push(i);
    } // 锁析构了
    cv.notify_one(); // 通知消费者有新产品
}

int pop_product() {
    std::unique_lock<std::mutex> lk(mtx);
    cv.wait(lk, [] { return !q.empty(); }); // 等待产品
    auto product = q.front(); // 取出产品
    if (product != 0) {
        q.pop();
    }
    return product;
}
```

在主函数创建2个生产者线程和若干个消费者线程，可以使用 `thread` (C++11)全部创建后再全部 `join` 使得主进程等待其他线程退出后再退出，对于消费者线程，统一创建一个 `std::vector<std::thread>` 维护，然而线程类型 `std::thread` 持有资源，不可拷贝，所以使用移动语义 `std::move` 移动进容器内，销毁原来的对象。

```cpp
int main() {
    std::thread producer1_thread(producer); // C++20 引入的线程类，自动 join
    std::thread producer2_thread(producer);
    std::vector<std::jthread> consumers_threads;
    for (int i = 0; i < 32; i++) {
        std::thread consumer_thread(consumer, i);
        consumers_threads.push_back(std::move(consumer_thread)); // 不移动的话就会变成拷贝，但线程不可拷贝，因为持有资源
    }
    for (int i = 0; i < 32; i++) {
         consumers_threads[i].join(); // 等待所有消费者线程完成
     }
     producer1_thread.join(); // 等待生产者线程完成
     producer2_thread.join();
    push_product(0); // 通知消费者结束
    return 0;
}
```

也可以使用 `jthread`(C++20)，就不需要手动 `join`，在退出作用域的时候自动 `join`

```cpp
int main() {
    std::jthread producer1_thread(producer); // C++20 引入的线程类，自动 join
    std::jthread producer2_thread(producer);
    std::vector<std::jthread> consumers_threads;
    for (int i = 0; i < 32; i++) {
        std::jthread consumer_thread(consumer, i);
        consumers_threads.push_back(std::move(consumer_thread)); // 不移动的话就会变成拷贝，但线程不可拷贝，因为持有资源
    }
    producer1_thread.join();
    producer2_thread.join();
    push_product(0); // 通知消费者结束
    return 0;
}
```
